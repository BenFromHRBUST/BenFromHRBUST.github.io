A-Lacked Number

English Solution

We know that for every $x$, that $x ^\land x=0$. So, we can use this character to find the lost number.

Time Complexity: $O(N)$; Space Complexity: $O(1)$.

中文题解

由于对于任意数 $x$，$x^\land x=0$。所以，我们可以利用这个性质找到缺失的数。

时间复杂度：$O(N)$；空间复杂度：$O(1)$。

C-Dice Sum

English Solution

It is clear that is a classical problem of Dynamic Programming. Also, it can be solved by Memorization Search.

We need to search for values at every position and not forget to do memorization. The part of memorization is the sum of all the numbers in front of the position and the index of the position.

Time Complexity: $O(N^3)$; Space Complexity： $O(N^3)$.

中文题解

显然该题是一道动态规划的经典题目，同时，该题也可以用记忆化搜索求解。

我们只需依次遍历搜索每个位置的数值，并做好记忆化即可。记忆化的部分是当前所有数的和以及当前是第几个数。

时间复杂度：$O(N^3)$；空间复杂度：$O(N^3)$。

D-Range Count Query

English Solution

Through the description, the answer of each query only relates to $X$. So we only need to care about the position of each $A_i$. We can store all the positions of $A_i$ into the corresponding vector. In other words, the corresponding vectors store all the position of $X$. For each query, using Binary Search to find the positions of $L$ and $R$, the difference is the number of $X$.

Time Complexity: $O(Qlong)$; Space Complexity: $O(N)$.

中文题解

由题意可知，每次询问的答案只与 $X$ 有关，所以我们只需关注每一个 $A_i$ 的位置即可。将所有 $A_i$ 的位置存入对应的 vector，$X$ 对应的 vector 内储存的即为 $X$ 的所有所在位置。对于每次查询对该 vector 进行二分找到 $L$ 和 $R$ 的对应位置，其差值就是 $X$ 的个数。

时间复杂度：$O(QlogN)$；空间复杂度：$O(N)$。

E-K-colinear Line

English Solution

As we all known, that two points determine a straight line. So we can travel all possible lines and check how many points are in the line. If the number of points larger than $K$, that means this line fit the bill. Notice, we need deduplication of lines. I used map to do that, the first dimensionality is a string type label of the line, which store all the points cross it, the second dimensionality record whether it was recorded.

Don’t forget special judgments.

Time Complexity: $O(N^3)$.

中文题解

两点确定一条直线，遍历所有的直线，判断剩下的点有多少在该直线上，若该直线上的点的个数大于等于 $K$，则该直线符合要求。需要注意的是，直线需要去重，我使用了 map 进行去重，第一维是 string 类型的标记，记录了该直线上的所有点，第二维记录是否记录过（map 中有则表示已经记录过）。

别忘了特判。

时间复杂度：$O(N^3)$。
